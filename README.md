Este projeto é uma base completa de filmes de terror desenvolvida em HTML, CSS e JavaScript, criada com o objetivo de ajudar qualquer pessoa a escolher o próximo filme para assistir sem ficar em dúvida. Inspirado e desenvolvido a partir da Imersão Dev da Alura + Google, o projeto reúne uma coleção organizada de títulos do gênero, apresentando informações como sinopse, ano de lançamento e outros detalhes relevantes. Com uma interface simples, visual e intuitiva, a aplicação permite navegar pelos filmes de forma prática e rápida, oferecendo uma experiência agradável tanto para fãs de terror quanto para quem está apenas em busca de novas recomendações.
